Accordion

Group of collapse elements, with single or multiple expanded at once.

MDN: <details>
MDN: <summary>

Note that support for having one <details> element open at a time (done with the name attribute) is newer than the <details> element itself, and not as widely supported. A separate implementation will be provided in the future to provide this functionality without cutting off support for older browsers.

Additionally, an approach using mask-image has been used to color the svg chevrons, and this may not be as widely supported across all browsers. A different approach using unicode characters or self-writing SVGs will be added in the future to ensure compatibility.

Structure

The accordion is made of multiple <details> elements (collapses) with a <summary> element (trigger) inside each, and a <div> element (content) that is displayed when the <details> is open.

A shared name attribute can be added to the <details> elements to make sure that only one of them is open at a time.

Variants

The accordion can be configured to have one or multiple items open at once.

By default, all items can be opened at once. To make sure that only one item is open at a time, add a name attribute to the <details> elements.
