Select

Semantic HTML select element.

MDN: <select>
MDN: <option>
MDN: <optgroup>

Examples

States

Select elements can be put in different states, such as disabled or loading, to indicate their current status.

The disabled and aria-busy attributes can be used for this purpose.

Item Grouping

Items in the select can be grouped to organize them better.

Use the <optgroup> element to group options together, and the label attribute to provide a label for the group.
